# Projectile Motion Simulation


This project simulates projectile motion with air resistance using Python.

## Features

* 3D projectile simulation
* Gravity and drag forces
* Euler numerical integration
* Visualization using VPython

## Physics Model

The projectile motion is modeled using Newton's Second Law.

Total force:

F = mg - kv

Acceleration:

a = g - (k/m)v

Velocity update:

v = v + a dt

Position update:

x = x + v dt
